# gravity-import-fb
Plugin that allows you to import specific columns of a CSV file into your Gravity Form
## For whom?
Plugin built especially for people that need to import leads from their forms in social platforms such as Facebook, Instagram, LinkedIn. 
## Getting started
1. Create gravity form in which field with id=1 is email (```type email```), id=2 is called full_name (```type single line text```), id=3 is called phone_number (```type single line text```)
2. When configuring the plugin, enter the ID number of the created form
3. Configure notifications and QR codes for the form
4. Create a new page and select "Insert FB ADS" as a template.
5. Disable indexing of the created subpage and secure it with a password!
6. In the Gravity Forms settings, enable access to the API (```settings-> rest API-> Enable access to the API```)
7. Enjoy!
